# Azure Platform Engineering

Most of my experience comes from enterprise projects where reusable infrastructure, automation, and operational consistency are critical. Since I can't share the infrastructure I've built professionally, I wanted to create a public project that reflects how I personally approach designing and organizing Azure platforms.

Everything in this repository is built from scratch without using any company-specific code or configurations. The goal isn't just to provision Azure resources, but to demonstrate how I think about organizing cloud infrastructure, separating reusable components, documenting design decisions, and building platforms that are easy to maintain over time.

I've built Azure infrastructure professionally for several years, but most of that work lives inside private enterprise repositories.

This project gives me a place to demonstrate the same engineering principles using publicly shareable code while continuously improving my Azure, Terraform, Kubernetes, and Platform Engineering skills.
